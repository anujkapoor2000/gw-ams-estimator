# GW Cloud AMS Engagement Estimator

**NTT DATA — Guidewire Practice**

Interactive estimator for Guidewire Cloud AMS engagements covering PC, CC, BC, Digital (Jutro), L2/L3 incident management, enhancements, 3-year planning, SLA/credits, AI capabilities, and KT planning.

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

Push to GitHub, import repo in Vercel. Framework auto-detected as Vite.

- Build command: `npm run build`
- Output directory: `dist`

## Tech Stack

- React 18 + Vite 5
- No external UI libraries — pure inline styles
- NTT DATA brand colours (#003087 blue, #E4002B red)
